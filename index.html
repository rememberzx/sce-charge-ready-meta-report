聚焦南加州 `SCE / Edison` 家充补贴场景，拆解同类服务商在 Meta 上的有效投放模式、优质素材角度，以及最值得借鉴的落地页承接结构。

研究日期：2026-03-12 市场：Southern California 重点：广告钩子 / 资格筛选 / LP 转化

核心判断

  1. 同类里最有效的冷流量模型，不是泛电工广告，而是 `补贴金额钩子 + 资格自查漏斗 + 一站式安装承接`。
  2. Meta 广告负责制造点击理由，真正拉高线索质量的是落地页里的 `资格筛选`。
  3. 最值得参考的组合是：Saiyan Electric 的漏斗模型、Amp Up California 的强过滤文案、Smart Plug EV 的施工展示。

提案方向建议

  • Meta 冷启动主打 `最高 $4,200 电箱升级补贴`，但正文要同步写清楚 `SCE / single-family / panel < 200A`。
  • 独立站不要先做大而全企业站，优先做 `补贴资格页 + 60秒问卷 + 结果分流页`。
  • 素材不能只有平面利益点，必须加入 `before / after`、`panel upgrade 现场`、`permit / inspection` 等真实施工证据。
  • 再营销阶段再强化 `案例、FAQ、评价、许可证与保险`，承接高意向用户。

已验证有效的素材模式

1. 金额钩子直接开门

典型表达：`Up to $4,200`、`FREE panel upgrade`、`$1,000 OFF EV charger installation`

为什么有效：普通 homeowner 不一定懂 `200A panel`，但一定看得懂“能省多少钱”。

2. 资格门槛反而提高质量

典型表达：`Edison customers only`、`No apartments`、`Most people do not qualify`

为什么有效：虽然牺牲一部分 CTR，但能显著提升线索纯度，尤其适合高客单安装服务。

3. 60 秒资格自查是最佳中间转化

典型动作：不直接让用户预约安装，而是先做 `4 个问题 / 60 秒筛选`。

为什么有效：用户心理负担更低，比“立即预约”更适合冷流量，且便于销售跟进分层。

4. 结果页必须给下一步

典型做法：符合补贴者推进预约，不符合者给 `自费折扣 / 基础安装方案`。

为什么有效：把资格页从“流失点”变成“分流器”，减少无效点击损耗。

5. 施工过程比纯平面更可信

典型素材：旧 panel、升级施工、线路、车位、验收、完工后充电。

为什么有效：服务类广告天然需要“看见专业度”，真实施工画面比抽象文案更能建立信任。

6. 一站式不是口号，要给证据

典型表达：`Permit handled`、`Inspection passed`、`Licensed / Bonded / Insured`。

为什么有效：用户买的不是设备，而是“把复杂流程一次办完的确定性”。

最值得借鉴的落地页结构

推荐承接路径

  1. `广告点击`
  2. `补贴资格页`
  3. `60 秒资格问卷`
  4. `结果页`
  5. `预约 / 电话 / 上门评估`

不建议直接用传统企业站首页接冷流量。用户不是来了解公司，而是来确认 “我能不能拿到补贴、值不值得现在做”。

首屏必须有的 6 个元素

  1. 强利益点：`最高 $4,200 电箱升级补贴`
  2. 强限制条件：`仅限 SCE 服务区独栋/联排住宅`
  3. 一站式价值主张：`电改 + 充电桩 + Permit + 验收 + 补贴协助`
  4. 低门槛 CTA：`60秒资格自查`
  5. 关键门槛提示：`200A / 25英尺 / 180天`
  6. 基础信任背书：`Licensed / Bonded / Insured`
